
### 📦 Sistema de Gestão de Stock em Python

Este projeto é um programa simples em Python para gerir o stock de materiais através de um menu interativo no terminal.

### 🧩 Funcionalidades

O sistema permite realizar as seguintes operações:

 1. Adicionar Material
- Permite inserir um novo material no stock.
- Solicita o nome e a quantidade inicial.
- Evita duplicação de materiais já existentes.

 2. Consultar Stock
- Permite verificar a quantidade disponível de um material específico.
- Informa se o material não existir.

 3. Atualizar Stock
- Possibilita adicionar ou remover quantidades de um material existente.
- Verifica se há stock suficiente antes de remover.

 4. Exibir Stock Geral
- Mostra todos os materiais registados e respetivas quantidades.

 5. Sair
- Encerra o programa.
